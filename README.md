# AI-Native Engineering Pack

This repository contains a curated documentation pack on how software engineering is changing in the age of AI, how AI should strengthen governance and operating discipline across the lifecycle, and how teams can adopt AI-supported engineering without losing quality, accountability, or architectural integrity.

`Claude Code` and `OpenAI Codex` appear throughout the pack as recurring reference examples. They are used not as the only relevant tools, but as representative examples of two important patterns in the current landscape:

- guided, workflow-oriented support for bounded engineering tasks,
- and broader, tool-connected support across multiple phases of the SDLC.
